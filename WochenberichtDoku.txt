Wochenbericht User Manual

Introduction

Das Wochenberichtstool wurde in Sommer 2020 in Rahmen eines SWTP-Projekts entwickelt, damit Zeiterfassungen
pro Studenten effizient aufgeteilt werden.
Das Tool ist ein Programm, das eine CSV Datei einliest und,
die Stunden von jede Person jeweils in ihr Blatt zuordornet.


Software Portability
    - best Performance auf Linux System mit LibreOffice
    - Der QuelleCode kann unter Windows oder Mac Os Rechner uebertragen werden.
      muss aber das Frontend angepasst werden un jeweils eine Taste an ihre entsprechende Function verbunden werden.

Wie legt man ein neues Projekt?

Einfach, indem man die erste CSV-Datei reinimportiert.

Wichtig

    - Die Blätter Team, home und Summary dürfen nicht gelöscht werden !!
    - Man darf keine weitere Sheets(Blätters) erstellen außer die, die vom Programm erstellt worden.

Software Funktion
 -Datei werden Wochenlich importiert
 -Merkt sich die Eröffnung zBsp: ertes Import oder nicht
 -Sheets werden automatisch erstellt
 -Prüft immer bei einem Import die Namen der Personen, die An das projekt teilnehmen
    und wenn eine neue Person in der CSV datei ist (heisst eine neue Person tretet das Projekt bei),fügt dann das Tool automatish die Person hinzu.
 -letzte importierte Woche kann gelöscht werden
 -eine Person oder mehere Personen können gelöscht werden
        -Seine gearbeite Studen bleiben Trotzdem im Projekt fest.
 -Das ganze Projekt kann auf NULL gesetzt werden
