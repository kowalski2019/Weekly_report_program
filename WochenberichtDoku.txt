Wochenbericht User Manual

Introduction

Das Wochenberichtstool wurde in Sommer 2020 in Rahmen eines SWTP-Projekts entwickelt, damit Zeiterfassungen
pro Studenten effizient aufgeteilt werden.
Wochenberichtstool ist ein Programm, das eine CSV Datei einliest und,
die Stunden von jede Person jeweils in ihr Blatt zuordornet


Software Portability
    - best Performance auf Linux System mit LibreOffice
    - the source Code kann on Windows or Mac Os Rechner uebertragen werden.
      muss aber das Frontend angepasst werden un jeweils eine Taste an ihre entsprechende Function verbinden.


Datein, die das Programm einlesen kann
    -csv Datei aus JiraPlatform

Erste Datei reinimportierien.

Important
    - Die Sheets Team, home und Summary dürfen nicht gelöscht werden !!
    - man darf keine weitere Sheets(Blätters) erstellen außer die, die vom Programm erstellt wurden.

Software Funktion
 -Datei werden Wochenlich importiert
 -Merkt sich die Eröffnung zBsp: ertes Import oder nicht
 -Sheets werden automatisch erstellt
 -Prüft immer bei einem Import die Namen der Personen, die An das projekt teilnehmen
    und wenn eine neue Person in der CSV datei ist fügt er die Person hinzu.
 -letzte importierte Woche kann gelöscht werden
 -eine Person oder mehere Personen können gelöscht werden
        -Seine gearbeite Studen im Projekt bleiben Trotzdem im Projekt fest.
 -Das ganze Projekt kann auf NULL gesetzt werden
